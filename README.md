이해람 온라인 과제
=============

### 서버실행
npm start


## 디렉토리구조
> API 폴더   
> > api.js   
> > github rest api를 사용한 함수를 정의해놓은 파일입니다.   
> > 
> js 폴더   
> >  common.js   
> >  범용적인 함수를 정의해놓은 파일입니다.   
> >  searchRepositories.js   
> >  저장소 검색에 관련된 함수를 정의해놓은 파일입니다.   
> >  searchUserList.js   
> >  유저 검색에 관련된 함수를 정의해놓은 파일입니다.   
> >  
> app.js   
> 메인 js 입니다.
